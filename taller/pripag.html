Tecnologías Web

La evolución de las tecnologías web ha venido acompañada de amplias ventajas.
Estas ventajas influyen tanto en desarrolladores como consumidores los cuales disfrutan el uso del web. Hablemos un poco de estas ventajas:

Ventajas:

  1. Gracias a la evolución de los lenguajes de programación:

    • Se pueden crear páginas más rápidas y potentes.

    • Se pueden realizar desiños más flexibles.

    • Nuevas herramientas de diseño.

  2. Las tecnologías web crean aplicaciones multiplaforma.

  3. Las aplicaciones se ejecutan en cualquier tipo de navegador web y sistema operativo.

  4. Las aplicaciones web ocupan solamente espacios de cache.

  5. Se actualizan automáticamente.


No siempre han existido estas ventajas, algunas se han ido mejorando gracias a la evolución tecnológica y otras en cambio se han ido agregando poco a poco a lo largo de la historia.

Esto quiere decir que ha habido cambios a lo largo de los años dentro del desarrollo de las tecnologías web.
Revisemos un poco la historia de las tecnologías web:

La web (World Wide Web o WWW) ha tenido una evolución sorprendente desde sus inicios. Esta evolución le ha permite convertirse en una herramienta muy poderosa.
Veamos aspectos claves de su evolución.

Inicios
  • La web se forma en 1989 gracias a Tim Berners-Lee.
  • Berners quería encontrar una manera de compartir información entre todos los científicos del mundo.
  • Crea la idea de ub sistema de hipertexto basado en Internet.
Primeros pasos
  • En 1991 Berners-Lee presentó el primer servidor web y cliente web.
  • El servidor permitía acceder y visualizar documentos por medio de hipervínculos.
  • Los documentos se escriben en HTML.
Popularizaron y Expansión de la web
  • La web se empieza hacer conocida y aparecen navegadores web, para facilitar la navegación y visualización.
  • La web se empieza a ver a nivel mundial.
Comercio electrónico y Actualizaciones.
  • Las empresas empezaron a ofrecer o promocionar sus servicios y productos en línea, creando la compra y venta electrónica.
  • HTML Version 4.01
  • Aparece la Web 2.0.
Redes sociales y creación de contenido
  • Aparecen sitios como Facebook, YouTube, Twitter, etc.
  • HTML versión 5.
  • JavaScript se comienza a implementar de manera fuerte.
Ultimos avances
  • Web 3.0
  • Web semántica.
  • HTTP/3.
